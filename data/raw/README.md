# Raw Data

This folder contains the original, unmodified datasets used in this capstone project.
All files are stored exactly as downloaded from official sources.
No cleaning, filtering, or transformation has been performed on these files.

## Datasets Included

### Dataset 1: Labour Force Participation
- Source: Statistics Canada
- Description: Contains information on labour force participation, employment, and unemployment trends in Canada over time.
- Purpose in CRISP-DM:
  - Phase 1 (Business Understanding): Understand labour market trends
  - Phase 2 (Data Understanding): Explore participation patterns and trends

### Dataset 2: Job Vacancies by Occupation
- Source: Statistics Canada
- Description: Provides data on unfilled job vacancies categorized by occupation.
- Purpose in CRISP-DM:
  - Phase 2 (Data Understanding): Identify occupations with high vacancy levels
  - Phase 3 (Data Preparation): Prepare vacancy indicators for analysis

### Dataset 3: Educational Attainment and Employment
- Source: Statistics Canada
- Description: Shows employment and unemployment outcomes by level of education.
- Purpose in CRISP-DM:
  - Phase 2 (Data Understanding): Analyze the relationship between education and employment
  - Phase 4 (Modeling): Assess impact of education on employment outcomes

### Dataset 4: National Occupational Classification (NOC)
- Source: Government of Canada (NOC)
- Description: Contains occupation classifications and associated skill levels.
- Purpose in CRISP-DM:
  - Phase 2 (Data Understanding): Understand skill and occupation structure
  - Phase 5 (Evaluation): Interpret results using occupational skill levels

## Data Integrity
Raw data files must not be modified. All cleaned and processed datasets are stored separately in the `data/processed/` folder.
